TherapyAgent S3 EC2 IAM Role Update

Changes:
- Supports EC2 IAM role credentials through IMDSv2/IMDSv1 fallback.
- Keeps existing `.env` access-key support for simple testing.
- S3 setup no longer requires AWS_ACCESS_KEY_ID/AWS_SECRET_ACCESS_KEY when EC2 metadata credentials are available.
- Adds credential_source in attachment setup JSON.
- Keeps SSE-KMS support through S3_KMS_KEY_ID.
- Repairs older review_history/files tables before index creation.

No npm install required.
